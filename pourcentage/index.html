Comment calculer un pourcentage ?

X % d'un nombre : multiplie le nombre par X puis divise par 100 (ex. 20 % de 80 = 80×20/100 = 16).
Proportion : pour savoir quel % une valeur représente, divise-la par le total puis multiplie par 100.
Variation : (valeur d'arrivée − valeur de départ) ÷ valeur de départ × 100, pour une hausse (+) ou une baisse (−).
